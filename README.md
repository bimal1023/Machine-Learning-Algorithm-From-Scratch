#  Machine Learning & Deep Learning From Scratch

This repository documents my journey of building core Machine Learning and Deep Learning algorithms completely from scratch — without using high-level libraries like scikit-learn or TensorFlow (except for verification).

The goal of this project is to deeply understand the mathematics, optimization, and implementation details behind ML and neural networks.

---

##  What I’m Building

- Linear Regression (Gradient Descent + Closed Form)
- Logistic Regression
- Gradient Descent Variants
- Neural Network from scratch (forward + backprop)
- CNN basics
- Optimization techniques
- Regularization methods
- Loss functions
- Activation functions
- And more as I learn

---

##  Why This Repository Exists

Instead of just using libraries, I want to:

- Understand the math behind each algorithm  
- Implement every step manually  
- Strengthen my fundamentals for research and graduate study  
- Build strong problem-solving and system-level thinking  

This repository reflects consistent learning and disciplined practice.
